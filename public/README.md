# Public Assets

Dieses Verzeichnis enthält die öffentlich zugänglichen Assets für die Website.

## Benötigte Dateien

### Logo
- **Dateiname:** `logo.png`
- **Pfad:** `/public/logo.png`
- **Verwendung:** Header-Logo
- **Empfohlene Größe:** Max. Höhe 60px, Breite automatisch

### Hero-Hintergrundbild
- **Dateiname:** `hero-background.jpg`
- **Pfad:** `/public/hero-background.jpg`
- **Verwendung:** Hintergrundbild in der Hero-Section
- **Empfohlene Größe:** Mindestens 1920x1080px für optimale Qualität

## Hinweise

- Alle Dateien in diesem Verzeichnis sind öffentlich zugänglich
- Verwenden Sie optimierte Bilder für bessere Performance
- Next.js serviert diese Dateien automatisch unter `/logo.png` und `/hero-background.jpg`
